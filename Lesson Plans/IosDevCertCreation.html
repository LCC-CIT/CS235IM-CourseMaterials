Adding an Apple Developer Certificate to XCode

Whether you are developing using Visual Studio with the Xamarin platform or using XCode you need a developer certificate from Apple.

  1. Create a certificate
    1. Go to the Apple Developer web site and log in.
    2. On the left side, click on "Certificates, IDs & Profiles"
    3. In the upper-right click on the plus button
    4. Select "iOS App Development", then click "Continue"
    5. The next screen gives instructions for creating a Certificate Signing Request (CSR). Read it and then open the Keychain Access app.
      1. For reference, this is the Keychain Access  launcher icon:


      2. In the Keychain Access drop down menu, select Keychain Access > Certificate Assistant > Request a Certificate from a Certificate Authority

      3. In the Certificate Information window, enter the following information:


      4. Save the CSR file to your computer. The file should be named "CertificateSigningRequest.certSigningRequest"
    1. Back on the Apple Developer web site,  click "Continue" and you will see this page:


      Choose the file you just created and click "Continue" to upload it.

    2. Now you can download the certificate to your computer. Yes, this seems kind of crazy! First you upload the signing request, then you download the certificate.


    3. After downloading the file open the Finder to the download location. The rest of the steps will be on your PC, not the web site.
      1. Double-click on the .cer file. This will open the dialog below:


        Click "Add".

      2. Now open XCode


      3. On the XCode menu, hen open "Preferences", then select the "Accounts" tab and click the + on the bottom left


        Select "Apple ID" and click "Continue.

      4. Enter your Apple ID and password


      5. Now your developer account will show up in XCode


That's it! Now you can use your developer license in XCode or Visual Studio.
